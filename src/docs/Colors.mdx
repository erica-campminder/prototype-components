# Colors

The CampMinder color system is built on six base palettes, each with a 100–1000 scale. Semantic tokens map these to roles like primary, error, and success.

## Semantic Tokens

These map base colors to roles. Use semantic tokens in components — not raw hex values.

<div style={{ fontFamily: 'Inter, sans-serif', display: 'flex', flexDirection: 'column', gap: 8, maxWidth: 600 }}>
  {[
    ['--color-primary-default', '#380F8C', 'purple-800', 'Buttons, links, key actions'],
    ['--color-information', '#2266CC', 'blue-600', 'Info states, badges'],
    ['--color-success', '#2B7C30', 'green-700', 'Success states'],
    ['--color-warning', '#EB991C', 'orange-700', 'Warning states'],
    ['--color-error', '#BE3000', 'red-700', 'Error states'],
    ['--color-page-background', '#F9FAFE', '—', 'Page background'],
  ].map(([token, hex, base, usage]) => (
    <div key={token} style={{ display: 'flex', alignItems: 'center', gap: 12, padding: '8px 12px', background: '#F9F9F9', borderRadius: 6 }}>
      <div style={{ width: 32, height: 32, background: hex, borderRadius: 4, border: '1px solid rgba(0,0,0,0.08)', flexShrink: 0 }} />
      <div style={{ flex: 1 }}>
        <div style={{ fontSize: 12, fontWeight: 600, color: '#282828', fontFamily: 'monospace' }}>{token}</div>
        <div style={{ fontSize: 11, color: '#808080' }}>{base} · {usage}</div>
      </div>
    </div>
  ))}
</div>
