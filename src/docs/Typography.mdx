# Typography

All type is set in **Inter**. The scale has five categories — Display, Headline, Title, Body, and Label — each with size, weight, and line-height defined as CSS custom properties.

---

## Display

Large hero text. Use for marketing, empty states, or major section headers.

<div style={{ fontFamily: 'Inter, sans-serif', display: 'flex', flexDirection: 'column', gap: 24, marginBottom: 40 }}>
  {[
    ['Display LG', '60px', 500, 1.133, '--type-display-lg'],
    ['Display MD', '48px', 500, 1.167, '--type-display-md'],
    ['Display SM', '36px', 500, 1.222, '--type-display-sm'],
  ].map(([name, size, weight, lh, token]) => (
    <div key={name} style={{ borderBottom: '1px solid #F0F0F0', paddingBottom: 16 }}>
      <div style={{ fontSize: size, fontWeight: weight, lineHeight: lh, color: '#282828', marginBottom: 8 }}>{name}</div>
      <div style={{ fontSize: 11, color: '#A0A0A0', fontFamily: 'monospace' }}>{token} · {size} · weight {weight} · lh {lh}</div>
    </div>
  ))}
</div>

## Using type tokens in CSS

```css
.my-heading {
  font-family: var(--font-family);
  font-size: var(--type-headline-lg-size);
  font-weight: var(--type-headline-lg-weight);
  line-height: var(--type-headline-lg-line-height);
}
```
